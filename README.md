# Виртуальное Путешествие в Средневековый Новгород

## Описание проекта

"Виртуальное Путешествие в Средневековый Новгород" - это интерактивная веб-карта города XI–XIII веков, представленная в виде изображения 'Новгород.png'. Приложение позволяет пользователям исследовать историческую карту с возможностью нажимать на различные точки интереса, заходить внутрь зданий (Новгородский кремль, дома жителей, церкви, дома ремесленников, княжеский дворец, торговые лавки) и наблюдать за жизнью бояр, князей и обычных жителей того времени. Вместо 3D-модели, реализована интерактивная карта-изображение, где пользователь может приближать участки, заходить в здания и видеть, как живут бояре, князи и обычные жители, читать, слушать рассказы, а также использовать голосовой ассистент для прохождения экскурсий по городу. Приближение и детализация элементов карты осуществляется через интерактивное масштабирование. Также в проекте присутствует голосовой ассистент, который проводит экскурсии по городу.

## Исторический контекст

Проект охватывает период XI-XIII веков, когда Новгород был одним из крупнейших политических, экономических и культурных центров Северо-Восточной Европы. В это время Новгород достиг наивысшего расцвета в рамках Новгородской республики, сохраняя значительную автономию в составе Киевской Руси.

### Ключевые исторические аспекты:
- Политическая структура: вече, посадник, тысяцкий, архиепископ
- Социальная структура: бояре, купцы, ремесленники, крестьяне
- Экономика: торговля с Ганзой, ремесла, земледелие
- Культура: летописание, иконопись, зодчество
- Религия: христианство, язычество, церковные институты

## Технологический стек

Для реализации проекта используется следующий технологический стек:

- **Интерактивная карта**: HTML, CSS, JavaScript
- **Язык программирования**: JavaScript
- **Формат изображения**: PNG ('Новгород.png')
- **База данных**: JSON-файлы или IndexedDB для хранения исторических материалов
- **Аудио**: Web Audio API, Web Speech API для голосового ассистента
- **Архитектурный подход**: модульная, гибкая архитектура с возможностью масштабирования

## Система навигации по карте

Приложение поддерживает интерактивную навигацию:
- **Клик по карте**: возможность выбрать точку интереса на изображении 'Новгород.png'
- **Переход внутрь зданий**: при клике на здание открывается информация о его содержимом и историческом значении
- **Просмотр жизни внутри**: возможность увидеть, как живут бояре, князи и обычные жители
- **Навигационные подсказки**: маркеры и указатели для удобства перемещения по карте

Система обеспечивает плавные переходы между видами и адаптивную навигацию в зависимости от типа устройства.

## Управление

- **Клик по зданию** - Открывает информационную панель
- **Кнопка "Зайти внутрь"** - Переход внутрь здания с отображением его интерьера
- **Кнопка "Выйти из здания"** - Возврат к общей карте
- **Клавиша ESC** - Закрывает информационную панель или выходит из здания
- **Кнопки +/-** - Масштабирование карты
- **Перетаскивание** - Перемещение по карте
